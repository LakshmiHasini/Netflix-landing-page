# Netflix Landing Page

This is a simple static landing page project (HTML/CSS) for a Netflix-styled page.

Files of interest:

- `netflix.html` — main page
- `src/css/style.css` — stylesheet
- `public/images/` — images used by the page

How to view locally

Option A — Open directly:
1. Open `netflix.html` in your browser (double-click or File → Open).

Option B — Serve over HTTP (recommended for assets):

```bash
# Python 3
python3 -m http.server 8000
# then open http://localhost:8000/netflix.html
```
